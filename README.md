# 🚗 Car Price Predictor

A Machine Learning web application that predicts used car prices based on various features like vehicle age, mileage, engine capacity, and more.

---

## 🔹 Features
- Predict car prices using ML model
- User-friendly interface built with Streamlit
- Real-time predictions
- Feature encoding and preprocessing

---

## 🔹 Tech Stack
- Python
- Streamlit
- Scikit-learn
- NumPy

---

## 🔹 Machine Learning Model
- Random Forest Regressor

---

## 🔹 How it Works
1. User enters car details
2. Data is processed and encoded
3. Model predicts estimated price
